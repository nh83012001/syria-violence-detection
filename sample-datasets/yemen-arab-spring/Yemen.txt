This file is an archive of Twitter data in CSV format generated by Deen Freelon on Saturday, February 11, 2012 from a set of tweets originally collected by the TwapperKepper sevice prior to March 20, 2011. The left column contains unique Twitter message IDs, while the right column contains unique Twitter user IDs. Information on how to retrieve a complete set of metadata for each tweet based on its unique ID can be found here: https://dev.twitter.com/docs/api/1/get/statuses/show/:id
